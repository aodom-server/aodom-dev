# aodom.dev Resume Site

Public QR resume site for hiring conversations and portfolio context.

## Stack

This is an Astro static site. Source files live in `src/`, static assets live in `public/`, and the generated site is committed to `site/` for deployment.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Astro writes the production output to `site/`.
